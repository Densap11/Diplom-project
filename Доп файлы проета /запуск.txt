**Запуск Для Демонстрации**

Проще всего показать проект уже в готовом виде через Docker.

1. Открой терминал в папке проекта:
```bash
cd "/Users/danilsac/Documents/Diplom project/backend"
```

2. Запусти сервисы:
```bash
docker compose up -d --build
```

3. Открой в браузере:
- пользовательский интерфейс: [http://localhost:3000](http://localhost:3000)
- документация API: [http://localhost:8000/docs](http://localhost:8000/docs)

4. Для показа с телефона в той же Wi‑Fi сети:
- [http://192.168.0.32:3000](http://192.168.0.32:3000)
- [http://192.168.0.32:8000/docs](http://192.168.0.32:8000/docs)

**Аккаунты**
- администратор: `admin@campus-demo.ru / password123`
- организатор: `organizer@campus-demo.ru / password123`
- студент: `student@campus-demo.ru / password123`

**Что Показать**
- под студентом:
  - просмотр общей доски
  - фильтрация мероприятий
  - запись на событие
  - профиль с моими записями
- под организатором:
  - создание события с обложкой
  - просмотр своих событий
  - редактирование
  - перевод в архив
  - удаление
  - просмотр участников с ФИО и телефоном
- под администратором:
  - управление пользователями
  - смена ролей
  - просмотр всех событий
  - управление статусами событий

**Если Нужно Остановить**
```bash
docker compose down
```

**Если Что‑То Не Открылось**
Проверь, что контейнеры запущены:
```bash
docker ps
```
